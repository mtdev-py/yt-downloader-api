# Privacy Policy — YT Downloader

**Last updated:** May 4, 2026

## Overview
YT Downloader is a Chrome extension that allows users to download YouTube videos as audio or video files. We are committed to protecting your privacy.

## Data Collection
**We do not collect, store, or share any personal data.** Specifically:

- ❌ No personal information (name, email, address)
- ❌ No browsing history
- ❌ No cookies or authentication data stored
- ❌ No analytics or tracking
- ❌ No user activity monitoring
- ❌ No location data

## How the Extension Works
1. The extension reads the YouTube video URL from your active browser tab.
2. The URL is sent to our backend server for processing.
3. The server downloads and converts the media file.
4. The converted file is sent back to your browser for download.
5. No data is retained on the server after processing.

## Data Transmission
- Video URLs are transmitted to our backend server solely for the purpose of downloading and converting media files.
- All communication is encrypted via HTTPS.
- No user data is logged or stored on the server.

## Third-Party Services
- The backend server uses **yt-dlp** (open-source) and **FFmpeg** (open-source) for media processing.
- No third-party analytics, advertising, or tracking services are used.

## Permissions Justification
| Permission | Purpose |
|---|---|
| `activeTab` | Detect YouTube video pages and read the URL |
| `tabs` | Query active tab URL for auto-fill |
| `cookies` | Display YouTube login status in the UI |
| `scripting` | Extract video metadata from the YouTube page |
